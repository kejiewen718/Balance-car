# From Scratch Workspace

这个目录不是对原参考工程的修改区，而是专门留给你“从 0 开始自己写一遍平衡车”的工作区。

## 使用原则

- 原工程保持不动，作为参考基线
- 新代码优先写在这个目录下
- 每完成一个阶段就提交一次
- 每一步都先验证输入输出，再进入闭环

## 建议目录

- `docs/`: 阶段计划、硬件记录、调参笔记
- `app/`: 主流程和任务调度
- `bsp/`: 板级支持包，时钟、串口、PWM、GPIO 等
- `modules/`: IMU、编码器、滤波、控制器等独立模块
- `experiments/`: 临时验证代码和测试记录

## 推荐推进顺序

1. 最小系统
2. 电机开环
3. 编码器测速
4. MPU6050 原始数据
5. 姿态解算
6. 角度环
7. 速度环
8. 转向环
9. 蓝牙 / OLED / 超声波

详细阶段说明见 [阶段检查表](docs/STAGE_CHECKLIST.md)。
