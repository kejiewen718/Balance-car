# 阶段检查表

## Stage 1: 最小系统

- [ ] 建立空白 Keil 工程
- [ ] 配置启动文件和系统时钟
- [ ] 跑通串口打印
- [ ] 跑通 LED 闪烁

## Stage 2: 电机开环

- [ ] 跑通 PWM 输出
- [ ] 跑通方向控制脚
- [ ] 左右轮正反转方向确认

## Stage 3: 编码器测速

- [ ] 左轮编码器读数正常
- [ ] 右轮编码器读数正常
- [ ] 正反转符号方向确认

## Stage 4: MPU6050 原始数据

- [ ] I2C 读写正常
- [ ] WHO_AM_I 正确
- [ ] 加速度原始值稳定
- [ ] 陀螺仪原始值稳定

## Stage 5: 姿态解算

- [ ] 倾角方向正确
- [ ] 静止角度稳定
- [ ] 角速度与倾角轴对应正确

## Stage 6: 角度环

- [ ] 纠偏方向正确
- [ ] 不会一上电暴冲
- [ ] 可在扶持状态下稳定纠偏

## Stage 7: 速度环

- [ ] 编码器反馈成功接入
- [ ] 积分限幅有效
- [ ] 静止时能收敛

## Stage 8: 转向环

- [ ] 左右转向控制生效
- [ ] 转向后不会立即失稳

## Stage 9: 外设与调试

- [ ] 蓝牙遥控
- [ ] OLED 显示
- [ ] 电池电压检测
